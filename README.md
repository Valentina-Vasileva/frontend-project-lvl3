RSS AGGREGATOR
==============
RSS Aggregator is the JS application for automatic collection from sources exporting to RSS formats.

### Installation

```console
$ git clone git@github.com:Valentina-Vasileva/frontend-project-lvl3.git
$ cd frontend-project-lvl3
$ make install
```

### Running locally

```console
$ make develop
```
